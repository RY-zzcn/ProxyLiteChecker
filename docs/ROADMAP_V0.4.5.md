## 1. 硬性约束

- [x] 保持原生 HTML/CSS/JavaScript、单 Go 二进制和 SQLite，不增加前端构建链或常驻外部服务。
- [x] 页面可观测数据在可见标签页内按秒级自动更新；隐藏标签页暂停高频请求，避免无意义负载。
- [x] 检测总并发默认 100，并在 API、持久设置和执行器三层限制为 1–300。
- [x] 前端保存并发设置后无需重启；运行中的检测任务可动态提高或降低并发，降低时不强杀已发出的请求。
- [x] 运行日志只保留最近约 30 条内存记录，不写入新的长期日志表，不随运行时间无限增长。
- [x] 只更新现有 `127.0.0.1:8899` 服务，不启动第二套临时部署。
